This program is developed under Code::Blocks 12.11 and runs on WIN32 platform.

It is recommended to install Code::Blocks 12.11 first, and then open Reversi.cbp for easy-compiling.

The AI algorithm uses MinMax tree with alpha-beta pruning. Especially , it has a priority next-step mechanism

to make alpha-beta pruning more efficient. The search depth is 8 by default and is possible to deepen 

if a more powerful CPU is available.

Furthermore, There is a benchmark option on the menu, which mainly test the CPU's capability for integer.
